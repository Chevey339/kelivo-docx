## 安装说明

- Android：在 Releases 下载 APK 安装到设备。
- iOS：通过上面的 App Store 或 TestFlight 链接安装。或自行签名安装。
- OpenHarmony：在 Releases 下载 hap 或 app 安装到设备。应用商店待上架。
